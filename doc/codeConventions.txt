These standards range from least to most specific.
They are all worth a read though much of the GNU C/C++ standard isn't applicable here.

The Eclipse auto-formatting tools can be used to (mostly) follow the gnu java standard for formatting.
However, more semantic structural guides have to be followed by you, the developer, manually.
Things can be fixed later, but doing things right the first time is often the best choice.
